# 字体设置
- 一个字、一句话、一个段落加上#就可以改变字体的大小
- 斜体：用左右各一个 *的方式包裹住文字就是斜体的语法
- 粗体:用左右各两个**的方式包裹住文字就是粗体的语法。

# 分割线
- 输入两个--
#  引用
- 引用：在内容首位加入>符号即可
